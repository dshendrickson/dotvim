# README

This is the repo for my vim dotfile management.

## Installation:

Clone the repo: `git clone git@github.com:geopet/dotvim.git`

CD into .vim: `cd .vim`

Run the vim-init.sh: `./vim-init.sh`

## Adding new plugins

1. Find vim plugin.
2. Add to vim-init.sh script
3. Rerun vim-init.sh
